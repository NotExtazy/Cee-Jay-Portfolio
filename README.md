# Cee Jay P. Madayag — Portfolio

A personal portfolio website showcasing my skills, projects, certifications, and contact information as a Full Stack Web Developer & Software Engineer.

## 🌐 Live Preview

Open `index.html` in your browser or visit the deployed site.

## 📁 Project Structure

```
Portfolio/
├── index.html      # Main HTML file
├── styles.css      # Stylesheet
├── script.js       # JavaScript (animations, form handling, EmailJS)
├── cj.png          # Profile photo
└── README.md       # Project documentation
```

## 🛠️ Built With

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, animations, responsive grid/flexbox
- **JavaScript (Vanilla)** — Intersection Observer, smooth scroll, EmailJS integration
- **EmailJS** — Contact form email delivery
- **Font Awesome** — Icons
- **Google Fonts** — Orbitron & Inter typefaces

## ✨ Features

- Responsive design (mobile, tablet, desktop)
- Animated hero section with particle effects
- Smooth scroll navigation with active link highlighting
- Contact form with EmailJS integration and bot honeypot protection
- Project showcase with tech tags and achievements
- Certifications and professional affiliations sections

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/NotExtazy/my-portfolio.git
   ```
2. Open `index.html` in your browser — no build step required.
